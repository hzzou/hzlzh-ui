## **Icon API**
### **Icon Attributes**

| 属性名   | 说明  | 类型            | 默认值    |
|-------|-----|---------------|--------|
| name  | 名字  | String        | ""     |
| color | 颜色  | String        | "#999" |
| size  | 大小  | String,Number | 16     |
